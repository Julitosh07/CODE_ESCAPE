# 🎮 CODE ESCAPE

## 🧩 Descripción
CODE ESCAPE es un videojuego en primera persona desarrollado en Unity.
El jugador despierta dentro de una habitación cerrada y deberá explorar el entorno, analizar objetos, completar retos y resolver acertijos 
para encontrar la forma de salir. El proyecto se encuentra en fase inicial, donde se ha construido la primera escena base con ambientación, objetos principales 
y perspectiva en primera persona.


## 🎯 Objetivo del Juego
El objetivo principal es resolver una serie de acertijos lógicos y de habilidad que permitirán desbloquear las puertas de los distintos
lugares de la casa misteriosa y avanzar hasta encontrar la salida. Cada nivel completado revelará un número que forma parte del código final necesario para escapar.

Cada vez que se avance en el juego se aumentará la dificultad y pondrá a prueba la observación y el razonamiento del jugador.

## 🏠 Escena Actual (Avance)

Actualmente el proyecto incluye:

- Habitación cerrada con paredes y piso
- Puerta bloqueada
- Cama y escritorio como objetos principales
- Vista en primera persona
- Ambientación básica

Esta escena representa el punto de inicio del juego.


## 🛠 Tecnologías Utilizadas

- Unity (Motor de desarrollo)
- C# (Próximamente para la lógica del juego)
- Modelos 3D importados de la AssetStore
